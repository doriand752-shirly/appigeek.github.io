Dépôt GitHub Pages « utilisateur » pour AdMob
============================================

Ce dossier est déjà un dépôt Git local (commit initial fait).

AdMob doit pouvoir lire :
  https://doriand752-shirly.github.io/app-ads.txt

1) Sur https://github.com/new créer un dépôt PUBLIC nommé exactement :
     doriand752-shirly.github.io
   Sans README ni .gitignore (dépôt vide).

2) Dans ce dossier, exécuter :

   git remote add origin https://github.com/doriand752-shirly/doriand752-shirly.github.io.git
   git push -u origin main

   (Si « remote origin already exists », utiliser : git remote set-url origin https://github.com/doriand752-shirly/doriand752-shirly.github.io.git)

3) Sur GitHub : Settings → Pages → Source : Deploy from a branch → Branch « main », dossier « / (root) ».

4) Vérifier https://doriand752-shirly.github.io/app-ads.txt puis dans AdMob : rechercher les mises à jour (jusqu’à ~24 h).

Ce dossier est listé dans .gitignore du dépôt appy-geek : il n’est pas poussé avec l’app, c’est volontaire (second dépôt Git).
