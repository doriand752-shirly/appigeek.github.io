AdMob — app-ads.txt
-------------------
Si le domaine « site développeur » dans App Store Connect / AdMob pointe vers
ce site projet, le fichier doit être accessible ici :
  https://doriand752-shirly.github.io/appigeek.github.io/app-ads.txt

Si AdMob exige le fichier à la racine du domaine utilisateur GitHub :
  https://doriand752-shirly.github.io/app-ads.txt
→ utiliser le dépôt séparé doriand752-shirly.github.io (voir historique du projet appy-geek).

Mise à jour du contenu
----------------------
Les pages HTML et app-ads.txt sont maintenues dans le dépôt appy-geek sous docs/.
Pour synchroniser : copier docs/index.html, docs/privacy.html et docs/app-ads.txt
vers ce dépôt puis git push.

GitHub Pages : Settings → Pages → Deploy from branch → main → / (root).
