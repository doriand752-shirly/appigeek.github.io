Pour que le site soit à la RACINE de appigeek.github.io (et pas sous
doriand752-shirly.github.io/appigeek.github.io/), le dépôt doit appartenir
au compte ou à l’organisation GitHub nommé exactement : appigeek

Étapes (une seule fois)
-----------------------
1. Créer une organisation GitHub : https://github.com/organizations/plan
   Nom : appigeek

2. Transférer ce dépôt vers l’org :
   GitHub → doriand752-shirly/appigeek.github.io → Settings → Danger zone
   → Transfer ownership → organisation appigeek
   (Le dépôt doit s’appeler toujours appigeek.github.io)

3. Dans appigeek/appigeek.github.io → Settings → Pages
   Source : Deploy from branch → main → / (root)

4. Attendre quelques minutes, puis vérifier :
   https://appigeek.github.io/
   https://appigeek.github.io/privacy.html
   https://appigeek.github.io/app-ads.txt
